Python

Introduction to Python

Python is a high-level, interpreted, general-purpose programming language.

  • Executes code line by line.
  • Handles memory management automatically.
  • Used in web dev, data science, AI, automation.
  • Simple, readable syntax with indentation.
  • Features a large standard library.

Variables & Data Types

Variables store data; Python is dynamically typed.

  • Numeric: int, float, complex.
  • Sequence: str, list, tuple.
  • Mapping: dict (key-value pairs).
  • Set: set (unique elements), frozenset.
  • Boolean: bool (True/False).

Operators

Symbols performing operations on values.

  • Arithmetic: + - * / % ** //
  • Comparison: == != > < >= <=
  • Logical: and or not
  • Assignment: = += -= *= /= etc.
  • Identity/Membership: is, in.

Control Flow

Determines execution order of code.

  • Conditional: if, elif, else for branching.
  • Loops: for (iterate sequences), while (repeat while condition is true).

Functions

Reusable blocks of code for specific tasks.

  • Definition: Use def.
  • Parameters: Inputs to the function.
  • Return Value: Result sent back using return.
  • Lambda: Small, anonymous functions.

Object-Oriented Programming (OOP)

Paradigm based on "objects" containing data and code.

  • Class: Blueprint for objects.
  • Object: Instance of a class.
  • Attributes/Methods: Object data and functions.
  • Inheritance: Class inherits from another.
  • Polymorphism/Encapsulation: Key OOP concepts.

Modules & Packages

Organize code for structure and reusability.

  • Module: A .py file with code.
  • Importing: Use import statement.
  • Package: Directory of modules (__init__.py).
  • Virtual Environments: Isolated environments for dependencies.

File I/O

Read from and write to files.

  • Opening: open() with modes ('r', 'w', 'a', etc.).
  • Use with open(...) for proper file closing.

Error Handling

Manage unexpected situations (exceptions) gracefully.

  • try: Code to monitor.
  • except: Catches specific exceptions.
  • finally: Always executes.
  • Use raise to trigger exceptions.

Advanced Topics (Brief)

  • Decorators: Modify function/method behavior.
  • Generators: Functions yielding iterators for memory efficiency.
  • Context Managers: Ensure proper resource handling (with statement).
  • Concurrency: Threading (within process) and Multiprocessing (separate processes).
  • Asynchronous Programming (asyncio): Non-blocking I/O with async/await.
